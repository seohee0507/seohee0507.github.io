@starting-style

요소 애니메이션 진입 시 첫 프레임 스타일

  • transition-behavior: ~s allow-discrete
    : 애니메이션 끝날 때까지(~s) 속성 전환 미룸
div{
	display:none;
	opacity:0;
	transition:
		opacity 0.3s,
		/* 애니메이션 끝날 때까지 display 속성 전환(display: none) 미룸  */
		display 0.3s allow-discrete;
}
@starting-style{
	div.active{
		opacity:0; /* .active 상태의 애니메이션 첫 프레임 스타일 */
	}
}
div.active{
	display:block;
	opacity:1;
}
