Framer Componenet Props

const MyComponent = ()=>{
	const [isValue, setIsValue] = useState(false);

	const someVar = {
		hover: (isValue) =>({
			scale: isValue? 1 : 1.5
		}),
		reset:{
			scale: 1
		}
	}

	return(
		<motion.div
			initial="reset"
			whileHover="hover"
			variants={someVar}
			custom={isValue}
			onClick={()=> setIsValue(true)}
		>
		</motion.div>
	)
}
  • initial: 초기상태
    • x, y, rotate …
  • animate: 목표위치
    • animate(element, {목표}, {duration})
      animate("div", {rotate: 90, x: 20 ... }, {duration:0})
  • transition: 전환 모습
    • tween : default, spring, inertia
    • ease, delay, duration
  • variants: 변수, 함수정의 → 객체 반환
    • 함수 정의 시 하나의 인수 받고 객체 반환할 수 있음
    • custom 통해 prop 전달
